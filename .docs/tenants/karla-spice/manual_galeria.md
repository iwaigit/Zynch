# Manual de Usuario: Gestión de Galería

Este documento explica cómo utilizar el módulo de **Galería** en el Panel de Administración de **Zynch by iwai**.

## 1. Acceso
Para acceder, inicia sesión como administrador y navega a la sección **Galería** en el menú lateral.

## 2. Características Principales
El sistema cuenta con optimización automática para facilitar tu trabajo:
*   **Capacidad de Subida:** Puedes subir fotos de hasta **50MB**.
*   **Compresión Inteligente:** El sistema reduce automáticamente el peso de la imagen a ~300KB sin perder calidad visible.
*   **Marca de Agua:** Se agrega automáticamente el texto configurable (por defecto "zynch.fun") para proteger tu contenido.
*   **Renombrado:** Las fotos se guardan con nombres secuenciales (`ZNF_01`, `ZNF_02`...) para mantener el orden.

## 3. Cómo Subir una Foto
1.  Haz clic en el botón grande **+ Subir Nueva Foto** o en el recuadro "Añadir".
2.  Selecciona la foto desde tu dispositivo (computadora o celular).
3.  Espera unos segundos mientras el sistema procesa, comprime y sube la imagen.
4.  ¡Listo! La foto aparecerá en la cuadrícula.

## 4. Gestión de Fotos
*   **Ver en Grande:** Haz clic sobre cualquier foto para verla en pantalla completa (Modo Lightbox) y verificar la calidad y marca de agua.
*   **Eliminar:** Si deseas borrar una foto, haz clic en el icono de papelería (🗑️) que aparece al pasar el mouse por encima. Deberás confirmar la acción.

## 5. Límites
*   La galería está diseñada para mostrar las mejores **24 fotos**. Si alcanzas el límite, deberás borrar algunas antiguas para subir nuevas.
